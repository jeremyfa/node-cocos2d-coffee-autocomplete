Class Detail

cc.Speed(action, speed)
Changes the speed of an action, making it take longer (speed > 1) or less (speed < 1) time.
Useful to simulate 'slow motion' or 'fast forward' effect.
Parameters:
{cc.ActionInterval} action
{Number} speed

Field Detail

Deprecated:
